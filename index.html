Lead signal · agents

Loop engineering is the clearest agent pattern in the batch

Signal: Several sources came back to the same point: an agent is not just a model call. It is a loop with tools, memory, stop rules, tests, and review. The hard part is not the while loop. The hard part is deciding when the work is actually done.

Why it matters: this is where useful agent products will differ. The prompt matters, but the surrounding loop decides whether the agent ships a working result or a confident mess.
